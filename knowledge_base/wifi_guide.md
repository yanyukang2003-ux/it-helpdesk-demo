# 公司 WiFi 连接指南

## WiFi 网络列表

公司提供以下无线网络:

- **Company-Work**: 办公网络，连接后可直接访问内网资源，需要使用公司账号登录
- **Company-Guest**: 访客网络，仅提供互联网访问，无法访问内网
- **Company-IoT**: 物联网设备专用，请勿手动连接

## 连接办公 WiFi (Company-Work)

1. 在 WiFi 列表中选择 "Company-Work"
2. 安全类型选择: WPA2-Enterprise
3. 用户名: 你的公司邮箱地址
4. 密码: 公司统一认证密码
5. 如提示证书验证，选择"信任"

## 常见问题

### WiFi 连接后无法上网
- 确认连接的是 Company-Work 而非 Company-Guest
- 尝试忘记网络后重新连接
- 确认密码未过期（密码 90 天强制更换）

### WiFi 密码是什么
- 办公 WiFi 使用个人公司账号登录，不是统一密码
- 访客 WiFi 密码请联系前台获取，每周更换

### 会议室 WiFi 信号差
- 尝试重新连接
- 如多个会议室均有问题，请提交工单报告
