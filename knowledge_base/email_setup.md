# 企业邮箱使用指南

## 邮箱基本信息

- 邮箱服务: Microsoft 365
- 邮箱格式: 姓名拼音@company.com（如 zhangsan@company.com）
- 网页版地址: https://outlook.office.com
- 单封邮件附件限制: 50MB
- 邮箱总容量: 50GB

## 首次登录设置

1. 打开浏览器访问 https://outlook.office.com
2. 输入你的邮箱地址（入职时 HR 会告知）
3. 输入初始密码（入职时 IT 会发送到你的手机）
4. 系统会要求你修改密码，新密码要求:
   - 至少 12 位
   - 包含大写字母、小写字母、数字和特殊字符
   - 不能与最近 5 次密码相同
5. 设置 MFA 二次验证（推荐使用 Microsoft Authenticator App）

## 在手机上配置邮箱

### iPhone
1. 下载 Microsoft Outlook App
2. 打开 App，输入公司邮箱地址
3. 输入密码并完成 MFA 验证
4. 授权完成后即可使用

### Android
1. 在应用商店下载 Microsoft Outlook
2. 操作步骤与 iPhone 相同

## 常见问题

### 忘记邮箱密码
- 方法一: 访问 https://passwordreset.company.com 自助重置
- 方法二: 联系 IT 帮助台，提供工号进行人工重置
- 重置后临时密码有效期 24 小时

### 无法发送大附件
- 单封邮件附件限制为 50MB
- 大文件请使用 OneDrive 共享链接
- 操作方法: 在 Outlook 中点击"附件" → "浏览此电脑" → 上传到 OneDrive → 自动生成共享链接

### 收不到外部邮件
- 检查垃圾邮件文件夹
- 确认发件人是否被加入了黑名单
- 如持续收不到，请提交工单由 IT 检查邮件网关
